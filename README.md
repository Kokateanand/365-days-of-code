## 🎯 About The Project

This repository serves as a personal archive for the **365 Days of Code** challenge, organized in partnership with the **Scaler Discord community**. The core objective is to solve one algorithmic problem every day, sharpening problem-solving capabilities, logical thinking, and programming expertise on the **InterviewBit** platform.

## ⚙️ Compilation & Run Guide

### Java Solutions
To run any Java program, navigate to the target subfolder and execute:
```bash
javac Main.java
java Main
```

### C++ Solutions
To compile and run any C++ program, navigate to the target subfolder and execute:
```bash
g++ Main.cpp -o main
./main
```
